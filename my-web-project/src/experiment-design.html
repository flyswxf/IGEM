实验一

植物乳植杆菌在模拟小肠环境中的耐受性及其定植生长能力

实验目的

研究植物乳植杆菌在模拟小肠环境中，特别是在含胆盐的条件下的耐受性及其定植生长能力。

实验材料

  • 菌株：植物乳植杆菌及其他常见乳酸菌（如乳酸乳球菌、双歧杆菌等）。
  • 胆盐溶液：模拟小肠中的胆盐浓度，使用不同浓度（如0.1%、0.3%、0.5%）。
  • 培养基：模拟小肠的培养基，pH范围为6.5-7.0，包含碳源、氮源、矿物质及适宜生长的其他成分。
  • 小肠模拟系统：模拟小肠的培养环境，包括pH调节设备、温控设备（37°C）、氮气保护设备。
  • 无菌设备：生物安全柜、离心机、无菌操作工具等。
  • 平板计数法、qPCR设备：用于菌落计数、定量分析。

实验步骤

  1. 菌株准备
    • 接种培养：将植物乳植杆菌及其他乳酸菌菌株分别接种到适宜的培养基中，培养24小时，确保菌株处于对数生长期。培养基为MRS（或相应适合乳酸菌生长的培养基）。
    • 菌株鉴定：通过显微镜检查、培养基特征确认菌株身份。
  2. 小肠模拟环境定植实验
    • 模拟小肠环境：准备一个模拟小肠环境的培养系统，设置pH值为6.5-7.0，温度为37°C，模拟小肠的厌氧或低氧环境，维持氮气流通。
    • 胃肠模拟处理：将接种的植物乳植杆菌及其他乳酸菌菌株通过模拟胃肠环境的过程（模拟胃酸条件，pH 2.0，持续2小时）后，接种到小肠模拟培养基中。

定植观察

接种后的菌株在小肠模拟培养基中的生长曲线（菌落计数、浑浊度变化）进行监测。每隔6小时取样，使用平板计数法（CFU/mL）进行活菌计数。

同时，使用qPCR检测菌株在小肠模拟培养基中的定植情况，使用菌株特异性引物。 观察菌株是否能够在模拟小肠环境中定植并持续生长，尤其是植物乳植杆菌与其他乳酸菌的生长表现差异。

数据分析

比较植物乳植杆菌与其他乳酸菌在小肠模拟环境中的定植能力，定量分析定植数量、持续时间等指标。

预期结果

定植能力：植物乳植杆菌在模拟小肠环境中定植能力较强，能够在胃肠模拟处理后在小肠环境中稳定生长，并持续占据小肠模拟培养基中的一定空间。而其他乳酸菌的定植能力可能较差，尤其在模拟胃酸处理后。这些结果可以支持植物乳植杆菌作为益生菌在肠道中生长定植的潜力。

实验二

测定CRISPR-Cas9线路的可行性

实验目的

测试植物乳植杆菌（Lactobacillus plantarum）工程菌在特定条件下（pH值约6.8且含有食物源）是否能够产生绿色荧光蛋白（GFP），并验证两者缺一时，绿色荧光蛋白的产生量是否显著减少或不产生。

实验材料

  • 菌株：植物乳植杆菌（Lactobacillus plantarum）工程菌株，已通过基因工程改造，能够表达绿色荧光蛋白（GFP）。
  • 培养基：
    • 标准MRS培养基：用于植物乳植杆菌的常规生长。
    • 食物源培养基：含有乳糖、葡萄糖或其他可用碳源的MRS培养基，用于模拟“进食”环境。
    • pH调整培养基：MRS培养基的pH调整至6.8，用于模拟特定的生长环境。
  • 设备：
    • pH计：用于调整和监测培养基的pH值。
    • 荧光显微镜：用于观察细胞内绿色荧光蛋白的表达。
    • 荧光分光光度计：用于定量绿色荧光蛋白的荧光强度。
    • 光密度计（OD600）：用于监测菌株的生长情况。

实验步骤

  1. 菌株准备与培养

    从植物乳植杆菌工程菌株中挑选适量的菌种，并接种到MRS培养基中，培养至对数生长期。将对数生长期的植物乳植杆菌接种到不同类型的培养基中。 培养基准备与pH调整：将MRS培养基分为四组：

    • 组1：标准MRS培养基，pH为6.8，含有可用碳源（如乳糖或葡萄糖），模拟进食条件且保持适宜pH值。
    • 组2：标准MRS培养基，pH为6.8，但不添加可用碳源（即无食物源）。
    • 组3：标准MRS培养基，pH不调整（约为6.0-6.3），含有可用碳源（即“进食”环境）。
    • 组4：标准MRS培养基，pH不调整（约为6.0-6.3），且不含可用碳源（即无食物源且不适宜生长）。
  2. 接种与培养

    将植物乳植杆菌工程菌接种到以上四种培养基中，初始接种浓度为10⁶ CFU/mL。在37°C下，使用摇床恒温培养48小时，定期取样。

    绿色荧光蛋白的检测：

    • 显微镜观察：取样后，用荧光显微镜观察细胞内的绿色荧光蛋白表达情况。
    • 荧光定量：在每个时间点，取1mL的样本，通过荧光分光光度计测定绿色荧光蛋白的荧光强度（Ex=488 nm，Em=509 nm）。
    • 生长监测：每4小时使用光密度计（OD600）测量培养液的光密度，监测菌株的生长情况。

数据分析：

  • 荧光强度分析：比较不同组中绿色荧光蛋白的荧光强度，评估食物源和pH对绿色荧光蛋白表达的影响。
  • 生长曲线分析：使用OD600数据绘制各组的生长曲线，比较不同条件下的生长速率。
  • 统计分析：对荧光强度数据进行方差分析（ANOVA），检验不同组间的显著性差异。

实验组与对照组设置：

  • 实验组1：植物乳植杆菌接种到pH 6.8且含可用碳源的培养基中。
  • 实验组2：植物乳植杆菌接种到pH 6.8且不含可用碳源的培养基中。
  • 实验组3：植物乳植杆菌接种到未调整pH且含可用碳源的培养基中。
  • 实验组4：植物乳植杆菌接种到未调整pH且不含可用碳源的培养基中（对照组）。

预期结果

绿色荧光蛋白的产生：

  • 在实验组1（pH 6.8且含有可用碳源的培养基）中，植物乳植杆菌的绿色荧光蛋白应该表现出最高的荧光强度，表明在特定的pH和食物源条件下，绿色荧光蛋白的表达最为活跃。
  • 在实验组2（pH 6.8且无食物源）中，虽然pH条件适宜，但由于缺乏碳源，绿色荧光蛋白的表达将显著低于实验组1。
  • 在实验组3（未调整pH且含可用碳源的培养基）中，虽然食物源存在，但由于pH值不在理想范围（可能导致菌株生长受限），绿色荧光蛋白的表达量应低于实验组1。
  • 在实验组4（未调整pH且无食物源的培养基）中，植物乳植杆菌的生长将受到显著限制，绿色荧光蛋白的表达量应接近零。

生长情况：

  • 实验组1中的植物乳植杆菌应表现出最佳的生长速率和最强的绿色荧光蛋白表达。
  • 实验组2中的菌株生长会受到一定限制，绿色荧光蛋白的表达量较低。
  • 实验组3中，菌株的生长受pH限制，可能略低于实验组1，绿色荧光蛋白的表达量也会较低。
  • 实验组4中的菌株生长将受显著抑制，几乎没有绿色荧光蛋白产生。

pH对菌株生长和GFP表达的影响：

  • 预计在pH 6.8的条件下，植物乳植杆菌将达到最佳生长状态，并且绿色荧光蛋白的表达最强。
  • 在pH过低或过高的环境中，植物乳植杆菌的生长和绿色荧光蛋白的表达将受到抑制。

总结：

通过该实验，我们可以验证植物乳植杆菌是否需要在pH 6.8且含有可用碳源的环境中才能产生绿色荧光蛋白。

实验三

测试工程菌的能否根据是否进食自动调节酶的产生量

实验目的

测试工程菌（植物乳植杆菌）在不进食时，绿色荧光蛋白（GFP）产生量较低；而在进食（例如提供特定食物源或刺激）时，绿色荧光蛋白的产生量显著增加。

实验材料

  • 培养基：
    • 基本培养基：不含特定食物成分（如碳源）的MRS培养基，用于不进食时的培养。
    • 进食培养基：含有特定食物源（如乳糖、葡萄糖等）的MRS培养基，模拟植物乳植杆菌的进食环境。
  • 设备：
    • 荧光显微镜：用于观察细胞内绿色荧光蛋白的表达情况。
    • 荧光分光光度计：用于定量绿色荧光蛋白的产生量。
    • 光密度计（OD600）：用于监测菌株的生长情况。

实验步骤

  1. 菌株准备与培养：

    2. 从植物乳植杆菌工程菌株中挑选适量菌种，确保其能够表达绿色荧光蛋白（GFP）。 将工程菌接种到基本培养基中（不含任何额外的碳源），并在37°C恒温培养24小时，直到其处于对数生长期。

  2. 培养基准备：
    • 基本培养基：含有常规成分的MRS培养基，不添加任何额外的碳源（如乳糖、葡萄糖等）。
    • 进食培养基：在MRS培养基中加入适量的乳糖或葡萄糖（例如1%乳糖或1%葡萄糖），模拟植物乳植杆菌的进食环境。
  3. 接种与培养：

      4. 将植物乳植杆菌工程菌接种到上述两种培养基中，初始接种浓度为10⁶ CFU/mL。 培养条件：在37°C恒温培养48小时，采样时间点为0h、1h、2h、4h、6h

  4. 绿色荧光蛋白的检测：
    • 显微镜观察：在每个时间点取样，观察绿色荧光蛋白的表达。使用荧光显微镜观察植物乳植杆菌是否能够在细胞内发出绿色荧光。
    • 荧光定量：每个时间点，取1mL的样本，通过荧光分光光度计测定绿色荧光蛋白的荧光强度（Ex=488 nm，Em=509 nm）。
    • 生长曲线分析：使用光密度计（OD600）监测菌株在两种培养基中的生长情况，绘制生长曲线。

数据分析

  • 绿色荧光蛋白的定量分析：比较实验组1和实验组2中绿色荧光蛋白的荧光强度差异，评估进食时（提供碳源）绿色荧光蛋白的生成量是否显著增加。
  • 背景荧光检查：通过对照组检查非转基因菌株的荧光强度，确保实验中的荧光信号来源于绿色荧光蛋白的表达。

实验组与对照组设置

  • 实验组1：植物乳植杆菌接种到不含额外碳源的基本培养基中。
  • 实验组2：植物乳植杆菌接种到含有特定食物源（如乳糖、葡萄糖等）的培养基中。
  • 对照组：未转基因的植物乳植杆菌接种到相同的培养基中（不表达绿色荧光蛋白），用于排除背景荧光。

数据分析

  • 绿色荧光蛋白表达量：使用荧光分光光度计数据分析绿色荧光蛋白的表达量，比较进食与不进食情况下的荧光强度差异。
  • 统计分析：通过t检验或方差分析（ANOVA）分析进食时与不进食时绿色荧光蛋白表达的显著性差异。

预期结果

绿色荧光蛋白产生量：

  • 在不进食时（实验组1），植物乳植杆菌工程菌的绿色荧光蛋白表达应较低，荧光强度相对较弱。
  • 在进食时（实验组2），添加乳糖或葡萄糖等碳源后，植物乳植杆菌的绿色荧光蛋白表达应显著增加。随着培养时间的延长，绿色荧光蛋白的荧光强度会逐步增强，表明进食刺激了绿色荧光蛋白的生成。

背景荧光检测：

    通过对照组的检测，背景荧光应接近于零，确认实验组中观测到的荧光强度是由绿色荧光蛋白表达引起的。

实验四

最终产物是否无毒

实验目的

测试植物乳植杆菌是否能够将TPA（对苯二甲酸）和EG（乙二醇）转化为无毒的最终产物，以评估其在环境保护或生物降解中的潜力。

实验材料

  • 菌株：植物乳植杆菌工程菌株，能够进行有机化合物的代谢转化。
  • 化学试剂：
    • TPA（对苯二甲酸）：用作实验的原料之一，存在于多种塑料（如PET）中。
    • EG（乙二醇）：作为另一种原料，与TPA共同参与转化反应。
    • 无毒最终产物：预计的无毒产物可能包括醇类、酸类或其他可以通过代谢途径转化为无毒物质的中间产物。
  • 培养基：
    • 标准MRS培养基，用于植物乳植杆菌的培养。
    • 含TPA和EG的培养基，以测试乳植杆菌在这些化学物质存在时的代谢能力。
  • 检测工具：
    • 高效液相色谱（HPLC）：用于分析TPA、EG及其转化产物。
    • 气相色谱-质谱联用（GC-MS）：用于检测最终转化产物的种类。
    • pH计：用于监测培养液pH值变化，以间接了解代谢过程中是否产生酸性或碱性物质。
    • 细胞毒性检测试剂盒：用于评估转化后的产物是否具有毒性。

实验步骤

  1. 菌株培养：

    从植物乳植杆菌中挑选适量菌种，并接种到MRS培养基中，培养至对数生长期。将对数生长期的菌液接种到含TPA和EG的培养基中。初始接种浓度为10⁶ CFU/mL。

  2. 培养基准备：
    • TPA和EG含量：在MRS培养基中加入一定浓度的TPA（例如1-2 g/L）和EG（例如1-2 g/L），以模拟TPA和EG对植物乳植杆菌的影响。
    • 对照组：设置无TPA和EG的培养基，观察植物乳植杆菌的生长情况和代谢特性。
  3. 培养与反应：

    将接种的菌液在37°C恒温摇床上培养48小时。每隔6小时取样，分析培养液中的TPA和EG的浓度变化，记录pH值和菌株的生长情况（OD600）。

  4. 转化产物分析：
    • 采用HPLC分析每个时间点的TPA和EG浓度变化，并检测是否有中间产物的生成。
    • 使用GC-MS分析可能的最终产物，包括醇类、酸类或其他无毒的代谢产物。
  5. 细胞毒性检测：

    取样并使用细胞毒性检测试剂盒，评估TPA和EG转化后的最终产物是否对细胞有毒性影响。与未转化的TPA和EG原料进行比较，判断转化后的产物是否安全。

数据分析

  • 代谢转化分析：根据TPA和EG的浓度变化及中间产物的检测结果，分析植物乳植杆菌是否能够有效转化TPA和EG。
  • 产物分析：通过GC-MS和HPLC数据，确认最终产物的种类，并分析这些产物是否为无毒物质。
  • 毒性评估：通过细胞毒性检测，比较转化后的产物和原料的毒性差异。

实验组与对照组设置

  • 实验组1：植物乳植杆菌接种到含TPA和EG的培养基中。
  • 对照组1：植物乳植杆菌接种到不含TPA和EG的培养基中（用于观察菌株在常规条件下的生长与代谢）。
  • 对照组2：植物乳植杆菌接种到仅含EG的培养基中。
  • 对照组3：植物乳植杆菌接种到仅含TPA的培养基中。

预期结果

TPA和EG的转化：

  • 预计植物乳植杆菌能够将TPA和EG转化为无毒的最终产物，可能生成醇类、酸类或其他无毒代谢产物。在培养过程中，TPA和EG的浓度会逐渐降低，表明这些物质被菌株转化。

代谢产物：

  • 在HPLC和GC-MS分析中，可能检测到如对苯二甲酸乙二醇酯（PET）降解产物、醇类或短链脂肪酸等无毒物质，证明植物乳植杆菌能够进行有效的代谢转化。

细胞毒性：

  • 通过细胞毒性检测，预计转化后的产物将不具备明显毒性，与未转化的TPA和EG相比，转化产物将显著降低其毒性。实验组中转化后的产物应该不具有致细胞毒性或表现出显著的细胞生存率。

生长情况：

  • 植物乳植杆菌在含TPA和EG的培养基中可能表现出与对照组不同的生长速率，尤其在TPA和EG作为唯一碳源时，菌株可能会逐渐适应并显示较慢的生长。

总结

通过本实验，我们可以评估植物乳植杆菌能够有效转化TPA和EG为无毒的最终产物。

实验五

LamB元件提高外源蛋白的分泌效率

实验目的

测试工程菌在表达LamB（外膜分泌蛋白元件）后，是否能够分泌更多的绿色荧光蛋白（GFP）到细胞外环境，从而提高外源蛋白的分泌效率。

实验材料

  • 菌株：
    • 实验组：携带LamB 元件的工程菌
    • 对照组：不含LamB 元件，仅表达GFP的工程菌
    • 阴性对照：野生型植物乳植杆菌（不表达GFP）
  • 质粒和构建：
    • pGFP质粒（控制组）：仅表达绿色荧光蛋白（GFP）
    • pLamB-GFP质粒（实验组）：带有LamB 分泌信号的GFP表达系统
  • 培养基：
    • MRS培养基：用于植物乳植杆菌的生长
    • MRS + 适当抗生素：筛选转化后的工程菌
  • 检测工具：
    • 荧光显微镜：观察细胞内与细胞外GFP荧光
    • 荧光分光光度计（Ex = 488 nm, Em = 509 nm）：定量分析培养液中GFP的荧光强度
    • 蛋白质定量检测：
      • Western Blot：检测细胞外GFP的分泌情况
      • 细胞裂解试剂（如Lysozyme酶解液）：破坏菌体，用于检测细胞内GFP水平
    • 离心机（8000 rpm, 10 min）：分离菌体和培养液

实验步骤

  1. 菌株培养

    将转化后的植物乳植杆菌（实验组、对照组）分别接种到MRS培养基中，37°C培养至对数生长期（OD600≈0.6）。

  2. GFP表达与分泌

    继续培养菌株，记录0h、1h、2h、4h、6h时的荧光蛋白分泌情况。取样后，8000 rpm离心10 min，分别收集：

    • 上清液（培养液） → 代表分泌到细胞外的GFP
    • 沉淀（菌体） → 代表细胞内GFP
  3. 细胞外GFP的定量分析
    • 荧光检测：使用荧光分光光度计测定上清液的荧光强度（Ex = 488 nm, Em = 509 nm），评估GFP分泌量
    • 蛋白浓度测定：进行Western Blot分析，检测分泌的GFP水平，并与标准GFP蛋白进行对比

数据分析

  • 细胞外GFP的表达量（上清液荧光强度）：
    • 比较实验组（+LamB）和对照组（-LamB）在不同时间点（0h、1h、2h、4h、6h）的上清液荧光强度
    • 预期结果：实验组GFP荧光强度高于对照组，表明LamB 促进GFP分泌到细胞外
  • Western Blot检测：
    • 细胞上清液GFP的Western Blot条带强度
    • 预期结果：实验组的上清液GFP条带比对照组更明显，说明分泌增强

总预期结果

LamB 元件增强了植物乳植杆菌的GFP分泌能力，使其在进食培养基中能够将更多GFP分泌到细胞外。
