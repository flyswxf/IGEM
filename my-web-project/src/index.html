降解微塑料、重建肠道环境的守卫菌

我们每个人每天都在摄入微塑料！！微塑料正在危害我们的身体！！

图1：外卖盒
图2：塑料袋
图3：塑料瓶

在现代社会，食物和水在进入人体前往往接触到塑料包装，微塑料（直径小于5毫米）随之进入体内。

研究表明，长期接触微塑料会对健康产生不良影响，其甚至能通过血脑屏障，并通过血液循环影响全身各部位。

为从源头减少微塑料对人体的危害，我们需要尽可能地阻止其被肠道吸收。微塑料还会破坏肠道菌群，使人体更难抵抗其危害。

查阅文献发现，乳酸菌中的植物乳植杆菌DT88能够像活性炭一样吸附微塑料。

如果我们赋予其降并让其在肠道定植，就能有解微塑料的功能，效重塑肠道环境。

产品设计

将工程菌设计成即食乳酸菌冻干粉末，把一次的量放在一个小瓶子里

人们可以选择空口吃或者随水/温度不超过40度的食物一同吃下。

这是我们平时的小肠，我们每天随着食物摄入的微塑料不仅会被小肠吸收，而且会破坏我们肠道内的益生菌肠道菌群，肠道环境的平衡被打破

当我们吃下工程菌之后，它作为乳植杆菌会在肠道内定植、补充肠道菌群，使我们的肠道环境恢复平衡

而且我们的工程菌会分泌微塑料降解酶，在小肠吸收微塑料之前，将其降解为无毒小分子
